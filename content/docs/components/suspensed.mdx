---
title: Suspensed
description: A thin wrapper around React.Suspense with a sensible default fallback.
---

<ComponentPreview name="suspensed" />

## Usage

```tsx
import { Suspensed, Spinner } from "@olwiba/ui";

// Default fallback is <Spinner />
<Suspensed>
  <LazyComponent />
</Suspensed>

// Custom fallback
<Suspensed fallback={<Spinner size="sm" />}>
  <LazyComponent />
</Suspensed>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Content to render (usually a lazy component) |
| `fallback` | `ReactNode` | `<Spinner />` | Shown while children are loading |

## Why

Writing `<React.Suspense fallback={<Spinner />}>` on every lazy import is repetitive. `Suspensed` defaults to `<Spinner />` so you only override when needed.
