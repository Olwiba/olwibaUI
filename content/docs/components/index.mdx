title: Components
description: App-level components provided by @olwiba/ui.
---

These are higher-level abstractions built on `@olwiba/cn` primitives. They solve common app patterns so you don't have to.

## App Primitives

| Component | Description |
|-----------|-------------|
| [ConfirmDialog](/docs/components/confirm-dialog) | Modal confirmation dialog — UI shell for the useConfirm hook |
| [EmptyState](/docs/components/empty-state) | Reusable empty state with icon slot, heading, description, and action |
| [FullPageSpinner](/docs/components/full-page-spinner) | Centered full-page loading state |
| [Kbd](/docs/components/kbd) | Keyboard shortcut badge — pairs naturally with RegisterHotkeys |
| [OlwibaUIContext](/docs/components/olwiba-ui-context) | Context provider for shared app state (isMobile) |
| [PageHeader](/docs/components/page-header) | Page title with description and action slot |
| [RegisterHotkeys](/docs/components/register-hotkeys) | Declarative global keyboard shortcut registration |
| [RootErrorFallback](/docs/components/root-error-fallback) | Full-page error boundary fallback |
| [Spinner](/docs/components/spinner) | Loading spinner with size variants |
| [Spotlight](/docs/components/spotlight) | Command palette shell with keyboard shortcut trigger and search input |
| [Suspensed](/docs/components/suspensed) | React.Suspense wrapper with default spinner fallback |
| [ThemeColorUpdater](/docs/components/theme-color-updater) | Syncs a color theme to `data-theme` on `<html>` |
| [ThemeSwitchMinimal](/docs/components/theme-switch-minimal) | Icon button to toggle dark / light mode |
| [VersionBanner](/docs/components/version-banner) | Dismissible banner for version announcements |

| Component | Description |
