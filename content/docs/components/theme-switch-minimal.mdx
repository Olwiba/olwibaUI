## Installation

```sh
bun add @olwiba/ui
```

## Usage

```tsx
import { ThemeSwitchMinimal } from "@olwiba/ui";
```

```tsx
<ThemeSwitchMinimal />
```

No props required. Reads and writes to `localStorage` under the `theme` key and toggles the `dark` class on `<html>`. Self-contained — no context or provider required.
