## Usage

```tsx
import { fireConfetti } from "@olwiba/ui";

// Fire from both bottom corners using theme colors
fireConfetti();

// Centered burst
fireConfetti({ centered: true, particleCount: 80 });

// Custom colors
fireConfetti({ colors: ['#ff6b6b', '#ffd93d', '#6bcb77'] });
```

## Origin

`fireConfetti` is implemented in `@olwiba/cn` and re-exported here for convenience. You can import it from either package.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `centered` | `boolean` | `false` | Fire from bottom-center instead of both corners |
| `particleCount` | `number` | `120` | Total particles |
| `spread` | `number` | `60` | Spread angle in degrees |
| `scalar` | `number` | `1.2` | Particle size scalar |
| `ticks` | `number` | `200` | Animation lifetime |
| `colors` | `string[]` | CSS vars | Colors to use — defaults to reading `--primary` and `--primary-foreground` from CSS |

## Theme colors

By default `fireConfetti` reads the active theme's `--primary` and `--primary-foreground` CSS variables, converts them from OKLCH to hex, and passes them to `canvas-confetti`. This makes the burst automatically match your active color theme.
