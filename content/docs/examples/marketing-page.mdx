## Usage

```tsx
import {
  MarketingNavBlock,
  MarketingHeroBlock,
  MarketingFeaturesBlock,
  MarketingStatsBlock,
  MarketingTestimonialsBlock,
  MarketingPricingBlock,
  MarketingCtaBlock,
  MarketingFooterBlock,
} from "@olwiba/ui";

export default function MarketingPage() {
  return (
    <div className="flex flex-col gap-4">
      <MarketingNavBlock />
      <MarketingHeroBlock />
      <MarketingFeaturesBlock />
      <MarketingStatsBlock />
      <MarketingTestimonialsBlock />
      <MarketingPricingBlock />
      <MarketingCtaBlock />
      <MarketingFooterBlock />
    </div>
  );
}
```

## Blocks Used

- `MarketingNavBlock` — Sticky top navigation with logo, links, and CTA
- `MarketingHeroBlock` — Hero section with headline and feature cards
- `MarketingFeaturesBlock` — Six-item features grid with icons
- `MarketingStatsBlock` — Bold metrics section
- `MarketingTestimonialsBlock` — Masonry quote cards
- `MarketingPricingBlock` — Three-tier pricing with billing toggle
- `MarketingCtaBlock` — Full-width call to action
- `MarketingFooterBlock` — Multi-column footer with social links
