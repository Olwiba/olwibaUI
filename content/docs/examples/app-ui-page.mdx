## Usage

```tsx
import { DashboardShellBlock } from "@olwiba/ui";

export default function AppPage() {
  return <DashboardShellBlock />;
}
```

## Blocks Used

- `MarketingNavBlock` — Top bar with logo and user actions
- `ApplicationSidebarBlock` — Left sidebar with grouped navigation
- `DashboardOverviewBlock` — KPI cards, chart, and data table
