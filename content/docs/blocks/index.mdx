---
title: Blocks
description: Pre-composed UI blocks built on @olwiba/cn primitives.
---

These blocks package common screen sections and can be dropped into app pages as starting points.

## Available Blocks

| Block | Description |
|-------|-------------|
| [AuthSplitBlock](/docs/blocks/auth-split-block) | Two-column sign-in section with feature callouts |
| [DashboardOverviewBlock](/docs/blocks/dashboard-overview-block) | KPI cards, invoice table, and progress panel |
| [DashboardShellBlock](/docs/blocks/dashboard-shell-block) | Full dashboard shell with inset sidebar, chart, and sortable data table |
| [MarketingHeroBlock](/docs/blocks/marketing-hero-block) | Hero section with CTA and feature cards |
| [ApplicationSidebarBlock](/docs/blocks/application-sidebar-block) | Full application sidebar with grouped navigation and workspace chrome |
| [DocumentSidebarBlock](/docs/blocks/document-sidebar-block) | Simple docs sidebar with collapsible navigation sections |
| [LoginBlock](/docs/blocks/login-block) | Centered login shell with muted background and compact sign-in form |
